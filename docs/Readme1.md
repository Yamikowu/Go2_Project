# RealSense D435 第一階段

這份文件整理目前已完成的功能、重複性分析，以及依「狗狗在我這 / 不在我這」分類可做的深度相機應用。

## 目前已完成

- `camera/demo/obstacle.py`：中央 ROI 距離判斷，前進/後退/待命（簡易避障/跟距）
- `camera/demo/smart_avoid.py`：左/中/右三區距離判斷，轉向/後退（進階避障）
- `camera/demo/follow_person.py`：找最近 blob + 距離控制 + 轉向（人物跟隨）
- `camera/demo/pointcloud.py`：深度轉點雲、3D 視覺化/存檔（掃描/建模）

### 重複性判斷與 `smart_avoid.py` 都是「避障」類；`smart_avoid.py` 多了三區域決策與轉向，層級更高。

- `follow_person.py` 是「追蹤人」，跟避障不同目的，但底層同樣使用深度距離。
- `pointcloud.py` 與其他三個功能完全不同，是「3D 掃描/建模」。

## 依情境分類（狗狗在我這 / 不在我這）

### A. 狗狗在我這（同場互動、即時控制）

適合示範給老師、同學看：

- **手勢控制**：推/拉、左右揮手、空中點按（深度即可）
- **人物跟隨**：`follow_person.py`（深度即可）
- **近距離安全層**：`obstacle.py` 或 `smart_avoid.py`（深度即可）
- **體感互動**：近距離「虛擬按鈕」觸發指令（深度即可）
- **快速 3D 掃描**：`pointcloud.py`（深度即可）

### B. 狗狗不在我這（遠端/自主模式）

適合做成「系統能力」與「長時間運作」的說法：

- **自動巡邏 / 隱私監控**：只用深度判斷物體/人出現與移動
- **防跌落偵測**：地面高度劇變就停下或後退
- **走廊置中 / 牆距維持**：靠深度維持與牆面的安全距離
- **室內建圖 / 掃描回傳**：點雲或簡化地圖回傳
- **動態避障**：移動中物體偵測與速度調整（Safety Layer 概念）

## 可擴充方向（可做但尚未完成）

- **人體姿態/火柴人**：完整骨架通常要 RGB 或外部模型（Mac 深度-only 可做簡化輪廓）
- **物體尺寸量測**：長寬高/體積估算（深度即可）
- **區域占用率/人數估計**：隱私友善、只看空間變化

## 一句話版本（報告用）

目前已完成「避障（簡/進階）、人物跟隨、點雲掃描」三大類；可再擴展到「同場人機互動」與「遠端自主安全」兩條線，符合深度相機的即時、安全、隱私友善特性。
